## 一、项目概述

一个轻量级的学生情绪/压力筛查 Web 工具。学生手机扫码填写 PHQ-8 + GAD-7 标准化量表（共15题），教师通过仪表盘查看全年级统计、需关注名单，支持 Excel 导出和数据备份。

**核心原则**：
- 非临床诊断工具，仅用于筛查和分流
- 学生匿名填写（仅学号），保护隐私
- 零服务器成本，全部使用免费云服务

## 四、数据流

### 学生提交流程

```
1. 学生输入学号 → 点击"开始填写"
2. 检查学号是否重复（去除前导零后比较）
   ├─ 重复 → 弹窗警告，学生可选择继续或取消
   └─ 不重复 → 直接进入问卷
3. 逐一作答 15 题（点击选项自动跳转下一题）
4. 全部答完 → 点击"提交"
5. 显示"提交成功"页面
6. 后台：GET /api → 获取现有数据 → 追加新记录 → PUT /api 写回
```

### 教师查看流程

```
1. 打开 dashboard.html → 输入密码
2. 密码验证（优先 localStorage，其次默认值 20260625）
3. GET /api → 获取全部提交记录
4. 客户端计算统计：
   - 总人数
   - 四级分布（正常/轻度/中度/中重度）
   - 需关注名单（PHQ-8 ≥ 10 或 GAD-7 ≥ 10）
5. 渲染 Chart.js 柱状图 + 学生表格
6. 每 5 分钟自动刷新
```

## 五、量表说明

### PHQ-8（患者健康问卷-8）

- 用途：抑郁症状筛查
- 题数：8 题（从 PHQ-9 删除第9题自伤筛查）
- 每题 0-3 分（完全不会=0 / 好几天=1 / 一半以上天数=2 / 几乎每天=3）
- 总分范围：0-24

| 分数 | 分级 |
|------|------|
| 0-4 | 正常范围 🟢 |
| 5-9 | 轻度抑郁倾向 🟡 |
| 10-14 | 中度抑郁倾向 🟠 |
| 15-19 | 中重度抑郁倾向 🔴 |
| 20-24 | 重度抑郁倾向 🔴 |

### GAD-7（广泛性焦虑量表-7）

- 用途：焦虑症状筛查
- 题数：7 题
- 每题 0-3 分
- 总分范围：0-21

| 分数 | 分级 |
|------|------|
| 0-4 | 正常范围 🟢 |
| 5-9 | 轻度焦虑 🟡 |
| 10-14 | 中度焦虑 🟠 |
| 15-21 | 重度焦虑 🔴 |

```bash

with zipfile.ZipFile('deploy.zip', 'w') as z:
    for f in ['student.html','dashboard.html','index.html','teacher.html','_redirects']:
        z.write(f)

  --data-binary @deploy.zip

# 3. 推送到 GitHub
git add -A && git commit -m "update" && git push github master
```

## 七、维护指南

### 日常维护

| 操作 | 频率 | 方法 |
|------|------|------|
| 查看数据 | 随时 | 打开仪表盘 |
| 备份数据 | 每次使用后 | 点「💾 备份数据」 |
| 保活数据 | 每 2-3 周 | 打开仪表盘刷新一次 |
| 清空数据 | 换新班级前 | 先备份 → 点「🗑 清空数据」 |
| 修改密码 | 需要时 | 点「⚙ 修改密码」 |

### ⚠️ 数据保活

jsonblob 是免费服务，超过 30 天无人访问可能自动清理数据。
**寒暑假前务必先备份！** 恢复方法：重新创建 jsonblob 端点并上传备份数据。

3. 等待 10 秒生效

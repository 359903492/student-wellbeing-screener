# 学生心理气象站 — 设计文档

**日期**：2025-06-25  
**状态**：已确认  
**用户**：教师（年级管理者）

---

## 1. 核心目标

为教师提供一个轻量级的学生情绪/压力筛查工具，帮助在期末等高压时期快速了解全年级学生心理状态，发现需要关注的学生并及时引导至专业渠道。

**非临床诊断工具**。不做诊断，只做筛查和分流。

---

## 2. 应用场景

- **场景**：期末复习期间，教师发现有学生以"抑郁"为由抗拒学习
- **使用方式**：教师投影/发送二维码 → 学生手机扫码 → 填写问卷 → 提交
- **规模**：全年级 300+ 人
- **频率**：每学期 1-2 次

---

- **前端**：纯静态 HTML + CSS + JS，部署到 **Gitee Pages**（国内访问稳定、免费）

- **数据库**：Google Sheets（存储原始答卷和汇总统计）

## 4. 测量工具

### 4.1 PHQ-9（患者健康问卷-9）

- **用途**：抑郁症状筛查
- **题数**：9 题
- **计分**：每题 0-3 分（完全不会=0 / 好几天=1 / 一半以上天数=2 / 几乎每天=3）
- **总分范围**：0-27
- **分级标准**（中国青少年常模验证）：
  - 0-4：正常范围 🟢
  - 5-9：轻度抑郁倾向 🟡
  - 10-14：中度抑郁倾向 🟠
  - 15-19：中重度抑郁倾向 🔴
  - 20-27：重度抑郁倾向 🔴

### 4.2 GAD-7（广泛性焦虑量表-7）

- **用途**：焦虑症状筛查
- **题数**：7 题
- **计分**：每题 0-3 分（同 PHQ-9）
- **总分范围**：0-21
- **分级标准**（中国青少年常模验证）：
  - 0-4：正常范围 🟢
  - 5-9：轻度焦虑 🟡
  - 10-14：中度焦虑 🟠
  - 15-21：重度焦虑 🔴

### 4.3 安全协议（PHQ-9 第 9 题）

PHQ-9 第 9 题："有不如死掉或用某种方式伤害自己的念头"。如果学生该题得分 ≥ 1：
- 仪表盘该学生条目标红色高亮
- 标注"⚠ 需立即关注 — 建议转介心理老师"
- 教师应在 24 小时内跟进

## 5. 页面设计

### 5.1 学生问卷页（student.html）

- **标题**："期末心理气象站 ☁️"（轻松校园风，不说"心理测试"）
- **UI 风格**：暖色系（橙/黄/米白）、圆角卡片、emoji 表情包装饰
- **结构**：
  1. 知情同意声明 + 班级/学号输入
  2. PHQ-9 板块（9 题），每题为一张卡片，大按钮选择
  3. GAD-7 板块（7 题），同上
  4. 提交按钮
  5. 提交后显示结果页：两条"温度计"（抑郁温度 + 焦虑温度）+ 分级 + 建议文案 + 心理老师联系方式
- **技术**：移动端优化（375-414px 宽度适配）、PWA 离线可用

### 5.2 教师仪表盘（dashboard.html）

- **访问控制**：URL 参数或简单密码验证
- **布局**（PC 端优先）：
  1. 顶部概览：四个数据卡片（良好/轻度/中度/中重度 人数+百分比）
  2. 中部：分班柱状图（Chart.js，堆叠柱状图，四种颜色分段）
  3. 底部：需关注学生表格（可排序、可筛选班级）
  4. 操作按钮：导出 Excel / 刷新数据
- **自动刷新**：每 5 分钟自动拉取最新数据

## 6. 数据隐私与伦理

| 措施 | 实现 |
|------|------|
| 匿名化 | 学生仅填学号（或自编代号），不填真实姓名 |
| 知情同意 | 问卷开头明确声明："结果仅班主任可见，不影响成绩" |
| 数据访问控制 | Google Sheets 仅授权给指定教师账号 |
| 非诊断声明 | 所有结果页面标注"本问卷为筛查工具，非临床诊断" |
| 安全协议 | PHQ-9 第 9 题高分自动标红，提醒立即转介 |
| 导出限制 | Excel 不含真实姓名，仅学号 + 班级 + 分数 |
| 知情同意文案 | "亲爱的同学：这份问卷帮助老师了解大家在期末阶段的情绪状态。你的回答只有班主任可见，不会影响任何成绩。请根据最近两周的真实感受作答。如果填完后觉得心情不好，随时可以找老师聊聊，或者联系学校心理老师。" |
